для клонирование рекомендуем использовать "$git clone --branch main --single-branch https://github.com/liyaFPV/nixos.git"

Внимание!!!!
для первой сборки системы рекомендуем с начало запустить install.sh а потом после успешной сборки flake-install.sh для избежание проблем 

первоя сборка может занимать час и более это нормально и не надо беспокоеться или остонавлевать прочесс 
рекомендуем не трогать пк и дождаться завершение устоновки

также если вы против закрытого софта то в app.nix в пораметре nixpkgs.config.allowUnfree вместо true впишите false но драйвера xepen будет не возможно устоновить и их надо убрать из списка програм для устоновки
